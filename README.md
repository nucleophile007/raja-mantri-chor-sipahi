## Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: TailwindCSS
- **Realtime**: Pusher
- **Database**: Upstash Redis
- **Video**: Remotion
- **Fonts**: Press Start 2P (Pixel vibes)

## Local Development (Kaise chalayein?)

1. Install dependencies:
```bash
npm install
# or
yarn install
```

2. Development server start karo:
```bash
npm run dev
```

3. Browser me kholo: [http://localhost:3000](http://localhost:3000)
